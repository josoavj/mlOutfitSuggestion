Limites connues et cas particuliers

Limites connues

Météo indisponible Si la météo n'est pas accessible, la pertinence peut baisser.
Profil incomplet Tailles ou morphologie manquantes réduisent la précision.
Styles hors vocabulaire Les styles non listés sont moins bien interprétés.

Cas particuliers

Agenda mixte Augmenter Top K pour couvrir plusieurs moments d'une journée.
Météo extrême Prioriser la météo avant les styles pour rester adapté.
Localisation absente En mode auto, la localisation est obligatoire.
